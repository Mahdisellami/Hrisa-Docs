## How to Use Bing

1. **Paste prompt** into the text box
2. **Click "Create"** button
3. **Wait 30-60 seconds** - it will generate 4 variations
4. **Review results** - Bing shows 4 different versions
5. **Click to enlarge** - Click any image to see full size
6. **Download** - Click download button (or right-click ‚Üí Save Image As...)
7. **Repeat** - Try another prompt!

## Tips for Best Results

### Getting 4 Variations
- Bing automatically generates 4 versions per prompt
- If you don't like any, click "Create" again with same prompt for 4 NEW versions

### Refining Results
After seeing results, you can modify prompts:
- Too complex? Add: "extremely simple, minimal details"
- Too cartoonish? Add: "professional, corporate, clean vector art"
- Wrong colors? Add: "exactly red #C62828 and white #FAFAFA"
- Need text? Add: "with text 'Hrisa Docs' in modern sans-serif font"

### Daily Limits
- Free tier: 15 fast generations per day (with "boosts")
- After boosts: Unlimited but slower
- Resets daily

## Troubleshooting

### "Content Warning" or Blocked
- Bing sometimes flags harmless prompts
- Try removing color codes: delete "#C62828" etc.
- Or simplify: "red chili pepper on white document"

### Results Too Realistic/Photo-Like
- Add to prompt: "flat design vector illustration icon style"
- Or: "not photorealistic, graphic design icon"

### Results Have Wrong Colors
- Be more specific: "bright red #C62828, pure white background"
- Or: "Tunisian harissa red color and white paper"

### Text is Garbled (if using text prompts)
- DALL-E sometimes struggles with text
- Generate without text first
- We can add "Hrisa Docs" text later in Figma/vector software
