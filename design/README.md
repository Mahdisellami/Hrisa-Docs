# Hrisa Docs Design Assets

This directory contains design references and assets for Hrisa Docs branding.

## Directory Structure

```
design/
├── references/          # Reference images for design inspiration
│   ├── documents/      # Document/paper icon references
│   ├── harissa/        # Harissa/chili pepper references
│   └── cultural/       # Tunisian cultural elements (Chechia, etc.)
├── concepts/           # Initial design concepts (to be created)
├── finals/             # Final approved designs (to be created)
└── assets/             # Production-ready assets (to be created)
```

## Design Brief

**Objective**: Create distinctive app icon and branding that combines:
1. Document processing theme (professional, academic)
2. Tunisian Harissa reference (cultural identity, bold/spicy)
3. Tunisian cultural elements (Chechia, traditional patterns)

**Key Requirements**:
- Works at all sizes (16x16 to 1024x1024)
- Distinctive and memorable
- Professional yet approachable
- Culturally authentic and respectful

**Color Palette**:
- Primary: Harissa Red (#D32F2F, #C62828, #B71C1C)
- Secondary: Document White (#FFFFFF, #FAFAFA)
- Accent: Tunisian Gold/Beige (#D4AF37, #C19A6B)
- Contrast: Dark Gray/Black (#212121, #424242)
